# SAP Automation

A React application I built for handling SAP automation tasks, specifically focusing on bank reconciliation statements (BRS). This project started as a way to streamline the reconciliation process for our finance team, and it's grown into a full-fledged tool with filtering, pagination, and data management features.

I spent quite a bit of time getting the filtering logic right – combining status filters with selection-based active/inactive toggles was trickier than expected, but it works smoothly now. The UI uses shadcn/ui components for a clean, modern look, and Tailwind for styling.

### Features

- Reconciliation table with advanced filtering (by status and selection)
- Pagination for large datasets
- Responsive design
- Login screen for authentication

### Getting Started

1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
